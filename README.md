## Overview
The To-Do List Engine is a command-line Python application that helps users manage daily tasks efficiently. The application allows users to add tasks, view all existing tasks, and maintain an organized task list through a simple menu-driven interface.

## Features
- Add new tasks
- View all saved tasks
- Automatic task ID assignment
- User-friendly menu system
- Input validation for empty tasks
- Lightweight command-line interface
